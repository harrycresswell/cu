Create unordered lists using the `ul` element:

- Unordered list item
- Unordered list item

```
<ul>
  <li>Unordered list item</li>
  <li>Unordered list item</li>
</ul>
```

Create ordered lists using the `ol` element:

1. Ordered list item
2. Ordered list item

```
<ol>
  <li>Unordered list item</li>
  <li>Unordered list item</li>
</ol>
```
