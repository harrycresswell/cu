Installation

There are a few different ways to install cu.css in your project, depending on whether you prefer to work with CSS or SASS.

Installing CSS:

  1. Download the full project from Github

  2. Copy public/css/main.min.css into your project.

  3. Copy static/fonts into your project.

  4. Download the full project from Github.

  5. Grab the CSS from Github

  6. Grab the CSS from Github

CSS

SASS

If you plan to work with the SASS files, either:

  1. download cu.css from Github and copy the SCSS folder into your project.
  2. Clone cu.css from GitHub git@github.com:harrycresswell/cu.git and copy the SCSS folder into your project.
  3. Install cu.css using a package manager: npm install cu-css
