Hero

Use .hero, combined with region and flow to create a distictive headline section for pages. For the sake of clarity, pipes are used to group classes, as seen below.

A modern CSS framework

cu.css is built using Cube CSS and Utopia fluid responsive design.

<div class="hero | region | flow">
  <h1>Some page title</h1>
  <p>Some page description</p>
</div>
