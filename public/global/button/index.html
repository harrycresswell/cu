Buttons

Use the <button> element to create interactive buttons. To style a button with the outline style, use the data attribute data-variant="outline". For more on this approach see Exceptionâ€™s in Cube CSS.

<button type="submit" value="Submit">Submit</button>
<button type="submit" data-variant="outline" value="Clear">Clear</button>
<button type="submit" disabled="">Disabled</button>

Inputs with type="submit", type="reset" or type="button" applied also inherit button styles. Therefore, you can apply the variant style to these elements too.

<input type="submit" />
<input type="submit" data-variant="outline" />

Just like with form elements, you can also use the w-full width utility class to make a <button> fill the full width of its parent container.

<button class="w-full" type="submit" value="Submit">Sign up</button>
